#  Панель администратора пиццерии

Административная панель для управления меню пиццерии. Позволяет добавлять новые позиции, редактировать существующие и удалять их из меню.

##  Функционал

### Основные возможности
- **Добавление новых пицц** в меню через интуитивную форму
- **Редактирование существующих позиций** (название, описание, цена, состав)
- **Удаление пицц** из меню 
- **Просмотр всего меню** в структурированном виде

### Управление данными пиццы
Каждая пицца содержит следующие поля:
- **Название** (обязательное поле)
- **Цена** (в рублях)
- **Изображение**
